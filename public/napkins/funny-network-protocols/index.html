I’m doing a side quest to find weird but useful network protocols.

  • Finger protocol. It used to be more popular decades ago. A finger command returns information about a user, including their login name, name, home directory, phone numbers, last seen, and other details. In 2020, Windows “finger” was used as living-off-the-land binaries (LOLBins) to retrieve information about users on remote computers running the Finger service or daemon.
  • Character Generator Protocol aka CHARGEN. A standard for the ARPA Internet community.
